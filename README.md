# GPU Specialization Capstone Project
## GPU Accelerated Image Edge Detection using CUDA

This project demonstrates GPU accelerated computing using NVIDIA CUDA.

The program runs a CUDA kernel that processes input data in parallel using GPU threads.

Features
- CUDA GPU parallel computing
- GPU memory allocation
- Host to device data transfer
- Parallel computation using CUDA kernels

Build

nvcc edge_detection.cu -o edge_detection

Run

./edge_detection

Output

GPU Edge Detection Completed
